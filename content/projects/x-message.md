# X-Message

## 项目介绍

X-Message 是一个信息聚合与开源项目展示平台，用于统一管理个人信息、开源项目和技术咨询。

## 功能特性

- 📱 响应式设计，支持PC和移动端
- 📝 Markdown文档实时渲染
- 💬 内置评论系统
- ⚙️ 配置文件化管理
- 📅 咨询按日期归档

## 技术栈

- Python Flask
- HTML5 + CSS3 + JavaScript
- Markdown

## 快速开始

```bash
pip install -r requirements.txt
python app.py
```

访问 http://localhost:8081
